软件工程管理 容国平 CMM/CMMI

CMM与CMMI

CMM1993年提出，现已弃用。

一种模式

一种评估模式，不是开发模式。是帮助改进开发的。

两种表现形式

阶段式

教条式的 maturity 有无

连续式

更灵活 capability 好坏

capability

maturity

做没做：maturity

做的结果：capability

三种不同的应用领域

系统工程（偏硬件，比如电器设备），人力资源管理。所有结构化描述的任务，及可以定义一个过程的工程，分成三种：

CMMI-DEV(CMMI for development)

CMMI-SVC(CMMI for Service)

  • SAAS: 一种软件发布和部署的方式。
  • XAAS: 比如PAAS，平台级服务。IAAS：基础设施及服务。

CMMI-SVC和CMMI-DEV在十六个过程域是相同的，但是它会更关注容灾和灾难恢复。

制约电子商务发展的重要因素是 信任度。

早期的电子商务产品是书。。

CRM：客户关系管理系统。

开源软件的质量，包括银行的很多底层软件都是开源的。这种开源更多的是一种服务。

CMMI-ACQ(采购)

系统集成：采购软硬件，组合，配置，形成一套解决方案。比如智能楼宇，智能家庭，给单位架设IT基础设施。这种适应于：重点不是开发而是选择正确的组件并正确的组合。

每一个项目不一定仅仅只包括上面的某一种。

同一个项目的客户有可能不同需求：

  • 客户希望系统在客户服务器上安装
  • 客户希望在公用服务器上安装
  • 客户希望合作搭建一个发布平台

三个模型共享十六个核心过程域

过程域是对最佳实践进行分组的结果。分成一组的实践相互间有紧密联系，是为了共同实现一个目的。

核心域在CMMI技术报告里。

四个类别

CMMI-DEV 大致分成两个类别：

  • 计划、跟踪：面向团队

    比如：配置管理。

  • 设计、编码、测试：面向这个项目

  • 改进

    比如工具引用

Humphry分成了四大类别：

项目管理类型 project management

估算、计划、跟踪

关注单个项目的实现

过程管理类型 process management

所有项目的过程方法（过程模型）

工程实践类型 engineering

支持类型 support

配置管理、度量和分析

一个过程域一定唯一的属于某个类别

四个级别

阶段式，maturity

一级：没有任何过程域

  • CMM：initial 初始级
  • CMMI：initial

没有规划，没有跟踪。没有任何评估的，开发比较混乱，没有计划；返工或救火盛行；项目成功或失败依赖个别程序员。或者计划不现实和粗略

！ 连续式的表现形式中单个过程域的第一级是performed（过程域中期望做的实践都做完了） 第零级： incomplete

二级

  • CMM中：Repitive 可重复级
  • CMMI：Managed 已管理级

什么叫管理：三个要素：有明确清晰的目标定义、有状态跟踪、纠正偏差的措施。即目标状态和纠偏。

有规划。在项目小组这一层有很明显的项目管理特征。做估算，计划和监控。并且可以反复使用。（可重复）

但救火文化依然盛行。

一级和二级代表了一种项目管理的方法：Reactive，反应式的，出了问题之后解决问题。

敏捷方法基本是二级。技术负债，只针对目前。但是不能绝对，因为重构是着眼未来的，考虑到未来可能会做的扩展。

三级

  • CMM中： defined 已定义级
  • CMMI中：defined 已定义级

Preactive

什么是defined：组织级有一套标准流程，每一个项目组根据当前特征在组织级的流程上定义(defined)、裁剪。

  • OSSP：组织级软件过程。
  • PDSP：项目组定义的软件过程。Program Defined software process.

发展动机：促成共享。不管是好的经验还是坏的经验。定义一个标准流程。EPG小组审核好的实践，提升为组织级标准流程的一部分。让分享变成必然。

四级

  • CMM中：Managed 已管理级
  • CMMI中：quantitive managed 已定量管理的

五级

  • CMM中：Optimizing 持续优化级
  • CMMI: Optimizing

第二章 链表问题

将搜索二叉树转化成双向链表

中序遍历，放入队列中，然后依次出队，再成链表。注意要新定义二叉树和双向链表的数据格式

package chapter_2_listproblem;
import java.util.*;
public class Problem_15_BSTtoDoubleLinkedList {
    static Node convert(TreeNode root){
        if(root == null) return null;
        Queue<TreeNode> q = new LinkedList<TreeNode>();
        inorder(q, root);
        Node head = new Node(q.poll().value);
        head.last = null;
        head.next = null;
        Node node = head;
        Node pre = head;
        while(!q.isEmpty()){
            TreeNode t = q.poll();
            node = new Node(t.value);
            node.last = pre;
            pre.next = node;
            node.next = null;
            pre = node;
        }
        return head;
    }
    static void inorder(Queue<TreeNode> q, TreeNode root){
        if(root == null) return;
        inorder(q, root.left);
        q.offer(root);
        inorder(q, root.right);
    }
    public static void main(String[] args) {
        // TODO Auto-generated method stub
        TreeNode t1 = new TreeNode(2);
        TreeNode t2 = new TreeNode(1);
        TreeNode t3 = new TreeNode(3);
        t1.right = t3;
        t1.left = t2;
        Node n = convert(t1);
        n.print(n);
    }
}

冒泡排序

class BubbleSort {
public:
    void swap(int &a,int &b)
        {
            int temp=a;
            a=b;
            b=temp;
    }
    int* bubbleSort(int* A, int n) {

            while(--n!=0)
             {

                    for(int j=1;j<=n;j++)
              {
                  if(A[j-1]>A[j])
                      swap(A[j-1],A[j]);
                  else
                      continue;
              }

            }

        return A;
    }
};

选择排序

class SelectionSort {
public:
    void swap(int &a,int &b)
        {
        int temp=a;
        a=b;
        b=temp;

    }
    int* selectionSort(int* A, int n) {
        int min;
        for(int i=0;i<n;i++)
            {
                min=i;
            for(int j=i;j<n;j++)
                {
                if(A[min]>A[j])
                    {
                    min=j;
                }
            }
            swap(A[i],A[min]);

        }
        return A;
    }
};

插入排序

申请额外空间的插入排序

package sort;
public class InsertionSort {
    static int[] insertionSort(int[] A, int n) {
        // write code here
        if(n ==0 || n==1) return A;
        int[] B = new int[n];
        int len = 1;
        B[0] = A[0];
        for(int i = 1; i<n; i++){
            int cur = A[i];
            int index = i;
            //获取A[i]应该插入的位置
            for(int j = 0; j<len; j++){
                if(cur<B[j]){
                    index = j;
                    break;
                }
            }
            //调整B数组
            for(int k = len; k>index; k--){
                B[k] = B[k-1];
            }
            //插入A[i];
            B[index] = cur;
            len++;
        }
        return B;
    }
    public static void main(String[] args){
        int[] A = {3,1,4,2,6,5,7};
        int[] B = insertionSort(A, 7);
        for(int i = 0 ; i<A.length; i++){
            System.out.print(B[i]+" ");
        }
        System.out.println();
    }
}

归并排序

package sort;
public class MergeSort {
     public static void mergeArray(int[] A, int begin, int mid, int end, int[] B) {
        int i = begin;
        int j = mid+1;
        int k = begin;
        while(i<=mid && j<=end){
            if(A[i]<A[j]) B[k++] = A[i++];
            else B[k++] = A[j++];
        }
        while(i<=mid) B[k++] = A[i++];
        while(j<=end) B[k++] = A[j++];
        for(int m = begin ; m<=end; m++){
            A[m] = B[m];
        }
    }
    public static void mergeSort(int[] A, int begin, int end, int[] B){
        if(begin<end){
            int mid = (end+begin)/2;
            mergeSort(A, begin, mid, B);
            mergeSort(A, mid+1, end, B);
            mergeArray(A, begin, mid, end, B);
        }
    }
    public static int[] mergeSort(int[] A, int n) {
        // write code here
        if(n == 0 || n == 1) return A;
        int[] B = new int[n];
        mergeSort(A, 0, n-1, B);
        return A;
    }
    public static void main(String[] args) {
        // TODO Auto-generated method stub
        int[] A = {1, 1};
        mergeSort(A, A.length);
        for(int i = 0; i<A.length; i++){
            System.out.println(A[i]);
        }
    }
}
