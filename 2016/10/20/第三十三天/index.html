38. Count and Say

The count-and-say sequence is the sequence of integers beginning as follows:

1, 11, 21, 1211, 111221, …

1 is read off as “one 1” or 11.

11 is read off as “two 1s” or 21.

21 is read off as “one 2, then one 1” or 1211.

Given an integer n, generate the nth sequence.

Note: The sequence of integers will be represented as a string.

这道题做出来了我还是比较满意的。毕竟第一次放弃了

解答：

public class Solution {
    public String countAndSay(int n) {
        if(n == 1) return "1";
        if(n == 2) return "11";
        if(n == 3) return "21";
        if(n == 4) return "1211";
        if(n == 5) return "111221";
        String[] tmp = new String[n+1];
        tmp[0] = "1";
        tmp[1] = "11";
        tmp[2] = "21";
        tmp[3] = "1211";
        tmp[4] = "111221";
        for(int i = 5; i<n; i++){
            String pre = tmp[i-1];
            String cur = "";
            int count = 1;
            int j = 1;
            for(;j<pre.length(); j++){
                if(pre.charAt(j) == pre.charAt(j-1)) count++;
                else {
                    cur += count;
                    cur += pre.charAt(j-1);
                    count = 1;
                }
            }
            cur += count;
            cur += pre.charAt(j-1);
            tmp[i] = cur;
        }
        return tmp[n-1];
    }
}

36. Valid Sudoku

Determine if a Sudoku is valid

借鉴了别人的部分代码，就是判断每个小格子是否合法的部分

解答：

class Solution {
public:
    bool isValidSudoku(vector<vector<char>>& board) {
        vector<unordered_map<char, int>> mapV;
        for(int i = 0; i<9; i++){
            unordered_map<char, int> map;
            vector<char> v = board[i];
            for(int j = 0; j<9; j++)
                if(v[i]!='.') map[v[i]] ++;
            mapV.push_back(map);
        }
        for(int i = 0; i<9; i++){
            unordered_map<char, int> map = mapV[i];
            for(int j = 0; j<9; j++){
                cout<<map[j+1]<<" ";
                if(map[j+1] >= 2) return false;
            }
            cout<<endl;
        }
        return true;
    }
};

39. Combination Sum

Given a set of candidate numbers (C) and a target number (T), find all unique combinations in C where the candidate numbers sums to T.

The same repeated number may be chosen from C unlimited number of times.

Note:

All numbers (including target) will be positive integers.

The solution set must not contain duplicate combinations.

For example, given candidate set [2, 3, 6, 7] and target 7, 

A solution set is: 
[
  [7],
  [2, 2, 3]
]

还是没做出来T_T

解答：

class Solution {
public:
    vector<vector<int>> res;
    int path[100] = {0};
    int len = 0;
    vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
        int n = candidates.size();
        if(n == 0) return res;
        robot(0, candidates, target);
        return res;
    }
    void robot(int idx, vector<int>& candidates, int target) {
        if(target == 0){
            vector<int> v;
            for(int i = 0; i<len; i++) 
                v.push_back(path[i]);
            res.push_back(v);
            return;
        }
        if(target<0 || idx >= candidates.size()){
            return;
        }
        path[len++] = candidates[idx];
        robot(idx, candidates, target-candidates[idx]);
        len--;
        robot(idx+1, candidates, target);
    }
};

第一章 js入门

js的对象和数组

对象

对象是名/值对的集合，或字符串到值映射的集合

var book = {
    topic: "JS",
    fat: ture
};

对象的访问：通过. 或者[]来获得

  • book.topic
  • book[“fat”]

属性的新建：.

  • book.author = “Flangon”; //通过赋值来新建一个属性
  • book.contents = {}; // {}是一个没有名字的空对象，它没有属性

数组

数组是以数字为索引的列表

var primes = [1,3,4,5];
primes[0]               //->1
primes.length           //=>4
primes[primes.length-1] //=>5
primes[4] = 9;  //通过赋值添加元素
primes[4] = 11; //通过赋值改变元素
var empty = []; //[]是空数组，它有0个元素
empty.length //=>0

对象与数组

数组和对象中都可以包含另一个数组和对象

var points = [{x: 0, y:0}, {x:1, y:1}];
var datas = {trial1: [[1,2], [3,4]], trial2: [[2,3],[4,5]]};

上面这种通过花括号定义对象元素和通过方括号定义数组元素的方法称为初始化表达式。

表达式是JS中的一个短语，这个短语可以通过运算得出一个值。

通过.或者[]来引用对象属性或数组元素就是一个表达式。

运算符

包括：

  • 1+2
  • 2-1
  • 2*1
  • 1/2
  • points.[1]+points.[2]
  • 字符串连接： “1”+”2” //=>“12”
  • 加减乘除等可以进行缩写

  • 相等关系

    通过相等运算符判断

    var x = 2, y = 3;
    x == y; // false
    "two" == "three"; //false
